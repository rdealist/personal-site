## 📝 添加新笔记(3步)

### 1️⃣ 创建文件
```bash
# 在 ai-glossary 目录创建新的 .md 文件
touch ai-glossary/新主题名称.md
```

### 2️⃣ 编写内容
```markdown
# 新主题名称

## 什么是XX?

**定义**: 简短说明

**详细内容**:
- 要点1
- 要点2

| 特性 | 说明 |
|------|------|
| 特性1 | 说明1 |

---

## 实践示例

\`\`\`python
# 代码示例
print("Hello AI")
\`\`\`
```

### 3️⃣ 自动发布
- 保存文件
- 刷新浏览器
- 笔记自动出现在 `/notes` 页面 ✨

## 🎨 支持的Markdown功能

✅ **基础格式**
- 标题 (`#`, `##`, `###`)
- 粗体 (`**粗体**`)
- 斜体 (`*斜体*`)
- 列表 (有序/无序)

✅ **高级格式**
- 表格
- 代码块(支持语法高亮)
- 引用块
- 链接和图片

✅ **GitHub Flavored Markdown**
- 任务列表 `- [ ]`
- 删除线 `~~删除~~`
- 自动链接

## 🔧 可选配置

### Front Matter(可选)

在笔记开头添加元数据:

```markdown
---
order: 1
description: "这是一篇关于XX的详细笔记"
---

# 笔记标题
...
```

- `order`: 控制笔记排序(数字越小越靠前)
- `description`: 自定义笔记描述(不设置会自动提取)
